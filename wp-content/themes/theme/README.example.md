## Browser support

* Front-end: https://www.gov.uk/service-manual/technology/designing-for-different-browsers-and-devices (assumed)
* Back-end: Unknown

## Development

### Install the dependencies:

```
composer install
yarn install
```

### Run the tests:

```
vendor/bin/kahlan spec
```

### Run the linter:

```
vendor/bin/php-cs-fixer fix
```
