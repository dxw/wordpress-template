Block that outputs a card that can be used for dynamic or static content

== Description ==

This card will output a title, image and content. 
If used in static form the user can add their own content using block editor components. 
If the dynamic option is chosed the user can select an article and the post will be populated with header, featured image and excerpt.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory and activate in wp-admin.
2. In the plugin root directory run `npm install`.
3. Run `npm start` to activate the build process if you are doing any development on the plugin.
